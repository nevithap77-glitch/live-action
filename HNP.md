Project Name: HNP LIVE AUCTION

Goal: Build a React website frontend using Ant Design (Antigravity) UI library, following dark purple and black theme (purple rays), fully responsive for mobile and desktop, and supporting future mobile apps. The website should implement user login, real-time auctions, product listings, bidding, group bidding, wishlist, and orders pages, with smooth animations, 3D effects, and form validation.

Pages & Features Required:

1️⃣ Homepage

Include logo (HNP LIVE AUCTION) and navigation bar

Navbar links: Home, Login, Product Listing, Bidding, Wishlist, My Orders, Group Bidding

Dark theme with purple and black gradient background (purple rays effect)

Responsive design (mobile + desktop)

2️⃣ Login Page

Form with username/email and password input

Login button with click animation

Form validation:

Check for empty fields

Show error messages

Dark purple theme with Ant Design Form & Input components

3️⃣ Product Listing Page

Display a grid/list of products with image + price

Include filters:

Price range

Category (High-Tech Gadgets, Collectibles, Luxury & Fashion, Experiences, Mystery Boxes, Art & Memorabilia)

Each product card must include “View Auction” button

3D hover effect on product cards when selecting

Smooth button click animations

Responsive design

4️⃣ Bidding Page (MAIN)

Display selected product with:

Image

Product details

Current highest bid

Countdown timer

“Place Bid” button

Bid history list (latest bids at the top)

Live updates via WebSockets (placeholder in frontend for now)

Loading spinner for async updates

3D effect for the product image

Form validation for bid input (non-empty, valid number)

5️⃣ Wishlist Page

Display saved items in a list/grid

Each item: image, price, and remove option

Smooth animations when removing items

6️⃣ My Orders Page

Display won items

Each item: image, price, and order status (Pending, Completed)

Responsive design

7️⃣ Group Bidding Page

Create Group Button

Invite Link for other users

Display members list

Contribution progress bar

“Place Group Bid” button

3D effect for selected product in group bidding

8️⃣ Product Types

High-Tech Gadgets

Collectibles

Luxury & Fashion

Experiences

Mystery Boxes

Art & Memorabilia

Focus on unique and vintage-style items to attract buyers

9️⃣ UI & Interaction Enhancements

Dark theme: purple + black with purple rays effect

3D hover effect on products

Button click animations

Form validation for all forms

Live update display placeholders for real-time bid updates

Loading spinner for async actions

Responsive design for mobile + laptop

Must support future integration with mobile applications

10️⃣ Extra Notes

Use Ant Design (Antigravity) for forms, buttons, cards, lists, modals, and notifications

Maintain consistent dark purple + black theme across all pages

Prioritize core functionality + frontend visuals + responsiveness

All pages must look professional and interactive for a hackathon demo
